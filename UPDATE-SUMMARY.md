## How to Update Your GitHub Repository

### Option 1: Replace All Files
1. Delete all files in your repository
2. Upload these new files
3. GitHub Actions will automatically redeploy

### Option 2: Update Individual Files
1. Click on each file in GitHub
2. Click "Edit this file" (pencil icon)
3. Copy content from updated files
4. Commit changes

The site will automatically rebuild within 1-2 minutes after any changes.
